Hexo-search配置

官方文档

安装插件

配置文件

修改在根目录下的_config.yml

  • path - 文件的路径。默认是 search.xml。如果文件扩展名为 .json,则输出格式为JSON。
  • field - 要搜索的搜索范围：
  • post (Default) - 仅涵盖您博客的所有帖子。
  • page - 仅涵盖您博客的所有页面。
  • all - 将涵盖您博客的所有帖子和页面。
  • content - 是否包含每篇文章的全部内容。如果false，生成的结果仅覆盖没有主体的标题和其他元信息。默认是true。

排除索引

在文章顶部配置里加入 indexing: false
