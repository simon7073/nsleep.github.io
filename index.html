1、安装Node.js和配置好Node.js环境
2、安装Git和配置好Git环境
git config –global user.name “your_name”
git config –global user.email “your_email”
ssh-keygen -t rsa -C “your_email” //创建ssh密钥，将公钥上传
eval “$(ssh-agent -s)” //添加密钥到ssh-agent
ssh -T git@github.com //测试添加ssh是否成功
3、Github账户注册和新建项目，项目必须要遵守格式：账户名.github.io
4、创个文件夹，然后通过命令行进入到该文件夹里面
5、安装Hexo
npm install hexo -g
hexo -v //检查hexo是否安装成功
6、hexo init //初始化该文件夹
7、npm install //安装所需要的组件
8、hexo g //首次体验Hexo
9、hexo s //开启服务器
10、hexo server -p 端口号 //改变端口号
11、在文件夹中，_config.yml文件，修改repo值（在末尾）
deploy:
type: git
repo: git@github.com:your_name/your_name.github.io.git
branch: master
12、生成以及部署文章之前，需要安装一个扩展
npm install hexo-deployer-git?–save
13、—————
14、hexo new post “博客名” //新建一篇博客–_posts目录下
15、hexo d -g //编辑器编好文章，那么就可以生成以及部署了
